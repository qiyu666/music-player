# 音乐播放器

一个功能丰富的在线音乐播放器，支持歌词同步、多种播放模式、快捷键操作等功能。

## 功能特性

- 🎵 基础播放功能（播放/暂停/上一首/下一首）
- 📝 歌词同步显示
- 🎧 多种播放模式（单曲循环/顺序播放/随机播放/列表循环）
- ⚡ 倍速播放（0.5x, 1.0x, 1.5x, 2.0x）
- 🔊 音量调节和静音
- 🔍 音乐搜索功能
- ❤️ 收藏/喜欢功能
- ✨ 当前播放歌曲高亮显示
- 🖱️ 歌词点击跳转功能
- ⌨️ 快捷键支持
- 📱 响应式设计，支持移动端
- 💾 使用localStorage保存用户设置

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| 空格 | 播放/暂停 |
| ← | 上一首 |
| → | 下一首 |
| ↑ | 音量+ |
| ↓ | 音量- |
| M | 静音/取消静音 |
| L | 打开/关闭播放列表 |
| S | 切换播放速度 |
| ? | 显示快捷键帮助 |

## 使用方法

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 开始享受音乐！
